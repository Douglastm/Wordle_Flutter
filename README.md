# 🎯 Termo Clone em Flutter

Um projeto desenvolvido em Flutter inspirado no jogo **Termo / Wordle**, onde o objetivo é adivinhar a palavra correta com base em dicas visuais.

## 🚀 Sobre o Projeto

Este projeto foi desenvolvido como parte de estudos em Flutter, com foco em:

* Gerenciamento de estado
* Manipulação de input do usuário
* Estruturação de projetos
* Criação de interfaces interativas
* Lógica de jogos

O jogo funciona de forma semelhante ao **Termo**, onde o usuário deve descobrir uma palavra de 5 letras com base nas seguintes regras:

* 🟩 Letra correta na posição correta
* 🟨 Letra correta na posição errada
* ⬜ Letra inexistente na palavra

---

## 🧠 Funcionalidades

* ✔️ Entrada de palavras pelo usuário
* ✔️ Validação de palavras reais
* ✔️ Feedback visual com cores
* ✔️ Sistema de tentativas
* ✔️ Interface baseada em grid
* ✔️ Animações simples nas letras

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone <url-do-repositorio>
```

2. Acesse a pasta:

```bash
cd nome-do-projeto
```

3. Instale as dependências:

```bash
flutter pub get
```

4. Execute o projeto:

```bash
flutter run
```

---

## 🎮 Como Jogar

1. Digite uma palavra de 5 letras

2. Pressione Enter ou clique no botão de envio

3. Observe as cores:

   * 🟩 Verde: letra correta no lugar certo
   * 🟨 Amarelo: letra correta no lugar errado
   * ⬜ Cinza: letra não existe

4. Descubra a palavra antes de acabar as tentativas!
